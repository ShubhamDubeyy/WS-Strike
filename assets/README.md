# Assets

This directory contains screenshots and images for the README.

## Required Screenshots

Add the following screenshots to complete the documentation:

- `ws-strike-banner.png` - Banner image (800x200 recommended)
- `history.png` - History panel screenshot
- `intercept.png` - Intercept panel screenshot
- `repeater.png` - Repeater panel screenshot
- `fuzzer.png` - Fuzzer panel screenshot
- `cswsh.png` - CSWSH/Hijack Test panel screenshot

## Creating Screenshots

1. Load WS-Strike in Burp Suite
2. Navigate to each tab
3. Capture with representative data
4. Resize to consistent dimensions
5. Save as PNG format

## Recommended Dimensions

- Banner: 800x200 pixels
- Panel screenshots: 1200x800 pixels
